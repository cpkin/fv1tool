Implement WAV file encoding and source code export:

**exportWAV.ts** - AudioBuffer to WAV encoder:
1. Function encodeWAV(buffer: AudioBuffer): Blob
2. Extract audio parameters: channels, sampleRate, length
3. Interleave channels into single Float32Array
4. Convert float32 samples (-1 to 1) to int16:
   - For each sample: clamp to [-1, 1]
   - Scale: sample < 0 ? sample * 0x8000 : sample * 0x7FFF
5. Build WAV header using DataView:
   - RIFF chunk descriptor: "RIFF", file size, "WAVE"
   - fmt sub-chunk: size 16, format 1 (PCM), channels, sampleRate, byteRate, blockAlign, 16 bits
   - data sub-chunk: "data", data size, samples
6. Write samples to DataView at offset 44 (after header)
7. Return Blob with type 'audio/wav'

8. Helper function writeString(view: DataView, offset: number, string: string)

9. Function downloadWAV(buffer: AudioBuffer, filename: string):
   - Call encodeWAV(buffer)
   - Create object URL
   - Create anchor element, set href and download attribute
   - Trigger click, revoke URL

10. Function downloadText(content: string, filename: string, mimeType: string):
    - Create Blob with content and mimeType
    - Create object URL, anchor click, revoke URL

**ExportButtons.tsx** - Export UI:

  <action>
Create URL hash encoding system for shareable state:

1. Interface URLState:
   - code: string (.spn source)
   - pot0: number (0-11 range)
   - pot1: number (0-11 range)
   - pot2: number (0-11 range)
   - demo?: string (optional demo ID)

2. Function encodeState(state: URLState): string
   - Serialize state to JSON: JSON.stringify(state)
   - Encode for URL safety: encodeURIComponent(json)
   - Base64 encode to compress: btoa(encodedJson)
   - Return hash format: `#state=${base64}`

3. Function decodeState(hash: string): URLState | null
   - Extract state parameter: hash.match(/#state=([^&]+)/)
   - If no match: return null
   - Base64 decode: atob(match[1])
   - URL decode: decodeURIComponent(decoded)
   - Parse JSON: JSON.parse(urlDecoded)
   - Validate structure (has code, pot0, pot1, pot2 fields)
   - Return URLState or null on error (catch JSON parse errors)

4. Function getCurrentState(code: string, pots: PotValues, selectedDemo: string | null): URLState
   - Convert POT values from 0.0-1.0 to 0-11: potN * 11
   - Build URLState object
   - Optionally include demo ID if selectedDemo is not null

5. Length safety:
   - Check encoded URL length before setting window.location.hash
   - If > 2000 chars: show warning "Code too large for URL sharing"
   - Consider LZ-string compression if needed (Claude's discretion for future enhancement)

Handle edge cases:
- Invalid base64: return null
- Malformed JSON: return null
- Missing required fields: return null
- POT values out of range: clamp to [0, 11]
  </action>

   - No separate Toast component needed

Handle edge cases:
- URL too long (>2000 chars): show error "Code too large for URL sharing (try shorter program)"
- Invalid hash on load: silently ignore, proceed with default state
